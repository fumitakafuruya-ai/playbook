# フィードバック

このプレイブックは**コミュニティで育てるドキュメント**です。  
あなたの声が、次の読者の助けになります。

## フィードバックの送り方

### GitHubアカウントをお持ちの方

各ページの下部にある**コメント欄**から直接書き込めます。  
ページごとにスレッドが分かれるので、該当ページで書くのがベストです。

### GitHubアカウントがない方

以下のどちらかでお気軽にどうぞ：

- **Facebookで直接メッセージ**  
  [古屋 文隆のFacebookページ](https://www.facebook.com/furuyafumitaka/)

- **GitHub Discussions**（アカウント作成は無料）  
  [Discussionsページ](https://github.com/fumitakafuruya-ai/playbook/discussions)

## こんなフィードバックを歓迎します

| 種類 | 例 |
|------|-----|
| 誤り・古い情報 | 「第3章のコマンドが動かなかった」 |
| わかりにくい箇所 | 「ここの説明が飛躍していて理解できなかった」 |
| 追加してほしい内容 | 「デプロイの章がほしい」「○○の使い方も知りたい」 |
| うまくいった報告 | 「第5章の手順で実際にアプリ作れました！」 |
| 改善提案 | 「この順番より○○の方がわかりやすいと思う」 |

どんな小さなことでも大歓迎です。  
「ここ、ちょっとわかりにくかった」——その一言が、このプレイブックをもっと良くします。
