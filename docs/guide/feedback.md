# フィードバック

このプレイブックは**コミュニティで育てるドキュメント**です。  
あなたの声が、次の読者の助けになります。

## フィードバックの送り方

各ページの下部にある**「付箋を貼る」**から書き込めます。  
アカウント登録は不要。お名前を入れて、TAV・感想・要望・その他から選んで貼るだけです。

もちろん、**Facebookで直接メッセージ**もお気軽にどうぞ：  
[古屋 文隆のFacebookページ →](https://www.facebook.com/furuyafumitaka/)

## こんなフィードバックを歓迎します

| 種類 | 例 |
|------|-----|
| TAV（気づき・学び） | 「この章で○○が腑に落ちた」 |
| 感想 | 「第5章の手順で実際にアプリ作れました！」 |
| 要望 | 「デプロイの章がほしい」「○○の使い方も知りたい」 |
| 誤り・古い情報 | 「第3章のコマンドが動かなかった」 |

どんな小さなことでも大歓迎です。  
「ここ、ちょっとわかりにくかった」——その一言が、このプレイブックをもっと良くします。

## :handshake: フィードバックから生まれたページ

実際に仲間からの声がきっかけで生まれたページを紹介します。

| きっかけ | 生まれたページ |
|---------|-------------|
| **けいちゃん**とのやりとり | [プロンプトの出し方](/guide/prompt-craft)（最重要ページ） — 抽象的な指示NG、1回1指示、小さく育てる3つの鉄則。サブスク内でやりきる課金の鉄則、失敗パターン集 |

| **いくちゃん**とのAI研究 | [第8章：MCPで外部サービスに繋げる](/guide/08-mcp) — Zoom・Gmail・Notionをひと言で操作する実践 |

**あなたの声が、次のページになるかもしれません。**
