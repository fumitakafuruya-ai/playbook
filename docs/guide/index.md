# このプレイブックについて

## 誰のために書いたか

**プログラミング経験ゼロの経営者**のために書きました。

筆者（古屋文隆）は51歳で初めてコードに触れました。Excelマクロの経験はありませんが、1993年からMicrosoft Accessでデータベースを作り続けてきたデータ好き。Power BIも使っていましたが、プログラミングは未経験でした。それが2週間後には8つのプロジェクトをGitHubで管理し、Webアプリを本番運用するまでになりました。

このプレイブックは、その道のりをそのまま体系化したものです。

## 前提条件

- パソコンが使える（Windows or Mac）
- ブラウザでWebサイトを見られる
- **プログラミング経験は不要**

## 何が得られるか

このプレイブックを終えると、以下ができるようになります：

| できること | 使うツール |
|-----------|-----------|
| コードを書く・読む | VS Code |
| 変更履歴を管理する | Git + GitHub |
| AIにコードを書いてもらう | Claude Code |
| Webアプリを作る | Next.js |
| 2台のPCで同じ環境で作業する | GitHub + Dropbox |
| 複数プロジェクトを一元管理する | VS Code ワークスペース |
