# 第2章：VS Codeを司令塔にする

## VS Codeとは

**Visual Studio Code（VS Code）** は、Microsoftが作った無料のコードエディタです。

「エディタ」と聞くと地味に感じますが、VS Codeは**開発の司令塔**です：

- コードを書く・読む
- Gitで変更を管理する
- ターミナル（コマンド実行）を使う
- Claude Codeと対話する

これが**1つの画面**で全部できます。

## インストール手順

### 1. VS Codeをダウンロード

1. https://code.visualstudio.com/ にアクセス
2. 「Download for Windows」をクリック
3. ダウンロードしたファイルを実行してインストール

### 2. 日本語化

VS Codeは初期状態では英語です。日本語にしましょう。

1. VS Codeを開く
2. `Ctrl + Shift + X` を押す（拡張機能パネルが開く）
3. 検索欄に `Japanese` と入力
4. 「Japanese Language Pack for VS Code」の「Install」をクリック
5. 右下に出る「Restart」をクリック

再起動後、メニューが全て日本語になります。

### 3. おすすめの初期設定

`Ctrl + ,`（設定画面）を開いて、以下を変更：

| 設定 | 値 | 理由 |
|------|-----|------|
| Auto Save | afterDelay | 保存忘れを防ぐ |
| Font Size | 14 | 読みやすいサイズ |
| Word Wrap | on | 横スクロールを防ぐ |

## 画面の見方

VS Codeの画面は4つのエリアに分かれています：

```
┌──────────┬────────────────────────────┐
│          │                            │
│ サイド   │     エディタ（メイン）       │
│ バー     │     ← ここでコードを書く    │
│          │                            │
│          ├────────────────────────────┤
│          │     ターミナル              │
│          │     ← ここでコマンドを実行   │
└──────────┴────────────────────────────┘
```

| エリア | 役割 | よく使うもの |
|--------|------|-------------|
| **サイドバー** | ファイル一覧、検索、Git | ファイルエクスプローラー |
| **エディタ** | ファイルの中身を表示・編集 | コード編集 |
| **ターミナル** | コマンドを実行 | Claude Code、npm、git |
| **ステータスバー**（最下部） | 現在の状態 | ブランチ名、エラー数 |

## ワークスペース：複数プロジェクトを一覧管理

プロジェクトが増えてきたら、**ワークスペース**が便利です。

1つのVS Code画面に複数のプロジェクトフォルダを並べられます：

```
サイドバー:
📚 EO Learning
📊 EO Survey
🏢 自社管理アプリ
📈 ライフライン
📅 業務管理アプリ
📱 Hibana
```

### ワークスペースの作り方

1. VS Codeを開く
2. 「ファイル」→「ワークスペースにフォルダーを追加」
3. プロジェクトフォルダを選択
4. 繰り返して全プロジェクトを追加
5. 「ファイル」→「名前を付けてワークスペースを保存」

次回からは `.code-workspace` ファイルをダブルクリックするだけで全プロジェクトが開きます。

## キーボードショートカット（最低限）

| ショートカット | 動作 |
|--------------|------|
| `Ctrl + S` | 保存 |
| `Ctrl + Shift + E` | ファイル一覧を表示 |
| `Ctrl + Shift + G` | Git（ソース管理）を表示 |
| `` Ctrl + ` `` | ターミナルを開く/閉じる |
| `Ctrl + Shift + P` | コマンドパレット（何でも検索） |

最初はこの5つだけ覚えれば十分です。
