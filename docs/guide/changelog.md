## v0.3 — 2026-04-09

### 新しい章
- **第7章：2台のPCで同じ環境を使う** を追加  
  Dropbox＋ジャンクションでClaude Code環境を複数PCで共有する方法

### サイト機能
- **「章から選ぶ」ナビゲーション** を追加 — どのページからでも目的の章にジャンプ可能に
- **フィードバックページ** を新設 — GitHubアカウントがなくても意見を送れる導線
- **giscusコメント欄** の案内を改善 — 非GitHubユーザー向けの誘導を追加
- **変更履歴ページ**（このページ）を追加

### 改善
- サイドバーを「基礎を整える」「作って動かす」「運用する」の3カテゴリに再編成
- ガイドトップページを章一覧ハブに改良（内容説明＋読了目安付き）
- 右側の目次がh3見出しまで表示されるように改善
- 各ページに「最終更新」日時を表示

### 技術的な変更
- VitePress を `1.0.0-alpha.28` → `1.6.4`（安定版）にアップグレード
- Vue を `3.2` → `3.5` にアップグレード
- OGPメタタグを追加（SNSシェア時のプレビュー対応）
- ローカル検索機能を有効化
