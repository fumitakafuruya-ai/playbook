## v0.5 — 2026-04-11

### 新機能
- **付箋コメント機能** を実装 — ログイン不要、名前を入れて貼るだけ
  - カテゴリ選択（TAV・感想・要望・その他）で色分け
  - Supabase + リアルタイム表示（貼った瞬間に全員に反映）
  - 投稿者本人のみ削除可能（ホバーで×ボタン表示）
  - メールアドレス欄（将来のアクセス制御用）
- giscus（GitHub必須コメント）を廃止、付箋に一本化

### サイトリブランド
- サイト名を **CC Playbook for CEOs** に変更
- トップコピーを「**Excelに限界を感じていませんか？**」に刷新
- ヒーローのサブコピー文字サイズを調整
- 鍵かけ予告の注意書きを追加

### コンテンツ更新
- **筆者プロフィール** — AIエージェントとの出会い（2/26 EO LALA）と開始日（3/2）を追加
- **Code Runner F** — 命名説明を簡潔化、匿名理由・引退宣言・登壇条件を追加
- **序章 Day 0** を追加 — 3/24サウナでの天啓、EO LALAメンバー3名への感謝
- **Hibana命名エピソード** — Claude Codeと特許庁調査から生まれた名前
- **EO Survey前史** — Gemini/Antigravityでの挫折とClaude Codeによる劇的改善
- **コスト表を刷新**（第1章） — 通常プラン$20〜/筆者プラン$135、白金ランチ比喩
- **第1章** — 「Power BIの限界」→「既存ツールの限界」に修正
- **第6章** — フォルダ構造の企業名を匿名化
- **第8章 MCP** — Chatwork追加（いくちゃん実装）
- **フィードバックページ** — GitHub説明削除、付箋での書き込みに一本化

### :handshake: フィードバック対応
- :handshake: **けいちゃん** — プロンプトの出し方ページ（最重要）
- :handshake: **いしいちゃん** — 章から選ぶナビゲーション（v0.3で対応済み）

## v0.3 — 2026-04-09
